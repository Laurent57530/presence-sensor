📡 Presence Sensor

Capteur de présence Zigbee — ESP32-H2 + LD2410S mmWave radar

Version : 1.0.0

⚠️ Utiliser Chrome ou Edge. Firefox n'est pas supporté.

🔌 Câblage LD2410S → ESP32-H2

LD2410SESP32-H2
Pin 13.3V
Pin 2GND
Pin 3GPIO4
Pin 4GPIO5
Pin 5GPIO10
⚠️ Ne pas utiliser GPIO9 sur le Super Mini — boot loop garanti.

🛠️ Procédure après flash

  1. Dans Zigbee2MQTT, activer "Autoriser l'appairage" puis mettre le device sous tension.
  2. Dans Z2M, page du device → onglet Lier → + Ajouter :
    Destination : Coordinator / Endpoint : 1 / Cluster : msOccupancySensing → Lier
  3. Dans Z2M, page du device → onglet Rapports → + Ajouter :
    Cluster : msOccupancySensing / Attribut : occupancy / Min : 1 / Max : 60 → ✅ valider → Synchro
  4. Vérifier dans l'onglet État : passer la main devant le capteur → {"occupancy": true}
  5. Dans Home Assistant, ajouter une carte Entité avec binary_sensor.xxx_occupancy.

⚙️ Comportement

  • Détection immédiate dès qu'une présence est captée
  • Délai d'absence : 30 secondes après la dernière détection
  • Détecte les micro-mouvements (respiration) — fonctionne même immobile
  • Portée : jusqu'à ~3 mètres

🔄 Ré-appairage (si nécessaire)

  1. Dans Z2M, page du device → supprimer le device (icône corbeille)
  2. Activer "Autoriser l'appairage" dans Z2M
  3. Redémarrer le device
  4. Refaire les étapes 2 et 3 de la procédure après flash
